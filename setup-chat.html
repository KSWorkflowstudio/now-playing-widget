🔗 Widget in StreamElements anlegen
1
Öffne StreamElements → Account → Channels
2
Kopiere deinen JWT Token (nicht den Overlay Token)
3
Token unten einfügen → „Widget anlegen" klicken
Wo findest du den JWT Token?
StreamElements → Account → Channels-Tab → Spalte JWT Token → Kopier-Symbol klicken
Der Token bleibt lokal in deinem Browser — er wird nicht gespeichert oder weitergegeben. → Zum SE Account
Das Widget wird direkt in deinem StreamElements-Account angelegt. Theme, Schriftgröße, Auto-Ausblenden und mehr kannst du danach im Fields-Tab in SE anpassen — ohne Code.
