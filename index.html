⚙️ Einstellungen in StreamElements ändern

      
Jederzeit in StreamElements → Widget bearbeiten → Fields-Tab — kein Code nötig:

      

        
lastfm_user / lastfm_keyLast.fm Zugangsdaten

        
themedark / frosted / light / minimal / neon / liquid

        
font_sizesmall / medium / large

        
show_bars / show_label / show_albumElemente ein-/ausblenden

        
custom_width / custom_radius / custom_blurGröße anpassen (0 = Standard)

      

      
Nach jeder Änderung OBS Browser Source neu laden (Rechtsklick → Interagieren → F5).

    


    

      
🔧 Fehlerbehebung

      

        
Widget bleibt leer
Benutzername + API Key prüfen. Scrobbling in der Musik-App aktiviert?

        
Kein Album-Cover
Last.fm-Profil muss öffentlich sein (Last.fm → Einstellungen → Datenschutz)

        
Design-Änderungen wirken nicht
OBS Browser Source mit F5 neu laden

        
Widget zeigt alten Song
Normal — Last.fm Scrobble-Verzögerung bis 30 Sekunden

        
Kein Frosted-Glass-Effekt sichtbar
Backdrop-filter braucht Hintergrundinhalt — OBS-Szene muss Inhalte hinter dem Widget haben
