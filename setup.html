🔗 Widget in StreamElements anlegen
1
Öffne StreamElements → Account und logge dich ein
2
Scrolle nach unten → klicke auf „Show Token" → Token kopieren
3
Token unten einfügen → auf „Widget anlegen" klicken
Der Token ist nur lokal in deinem Browser aktiv — er wird nicht gespeichert oder weitergegeben. → Zum SE Account
Das Widget wird direkt in deinem StreamElements-Account angelegt. Alle Einstellungen (Theme, Animation, Last.fm API Key usw.) kannst du danach jederzeit im Fields-Tab in SE ändern — ohne Code.
